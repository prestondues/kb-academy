# KB Academy Certification Workflow

## Purpose
Move certification quizzes into a trainer-led operational workflow.

## Flow
1. Trainer opens Certifications
2. Trainer selects trainee
3. Trainer selects module
4. System verifies quiz exists for the module
5. System starts or resumes certification quiz attempt
6. Trainer administers quiz with trainee present
7. Quiz is scored
8. If passing criteria is met, both trainer and trainee sign with PIN
9. Certification is finalized

## Notes
- Certification quizzes are not self-service
- Quiz attempt data supports scoring/history
- Certification status is finalized only after dual signoff
