# KB Academy Training Quiz Runner

## Purpose
Allow a trainee to take a quiz tied to a training module.

## Flow
1. Open quiz runner
2. Start new attempt or resume in-progress attempt
3. Answer questions by type
4. Submit quiz
5. Score is calculated
6. Pass/fail is stored in quiz attempts

## Supported Question Types
- multiple choice
- true / false
- scaled

## Notes
- answers save to the attempt record
- score is based on exact match against the correct answer set
- result screen shows score and pass/fail after submission
