# KB Academy Training Coverage Targets

## Purpose
Track how many people should be certified for a module within a department and shift, and compare that against current certified counts.

## Coverage Key
- module
- department
- shift

## Values
- target_count
- actual certified count
- gap = target_count - actual certified count

## Current Use
This is the foundation for workforce coverage planning and skill gap visibility.

## Future Expansion
- position-based targets
- expiring soon impact on coverage
- coverage dashboard widgets
- auto alerts for gaps
