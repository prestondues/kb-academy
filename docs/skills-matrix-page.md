# KB Academy Skills Matrix Page

## Purpose
Provide a visual qualification matrix showing which employees are currently trained or certified on which modules.

## Current Capabilities
- users listed down the left side
- modules listed across the top
- matrix cells display current qualification status
- supports filtering to keep the matrix readable

## Current Status Logic
- Current: certification exists and is not expired
- Expired: certification exists but has expired
- Missing: no certification exists

## Future Expansion
- in-progress state
- expiring soon state
- quiz-required state
- required hours / time progress
- department and shift filters
- click-through to detailed record history
