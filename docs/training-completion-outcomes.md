# KB Academy Training Completion Outcomes

## Purpose
Completed training sessions should create meaningful downstream records.

## Current Outcomes
- training session stores duration_minutes
- certification record created or refreshed for trainee/module
- expires_at is based on module recertification settings when applicable

## Certification Rules
- one certification record per trainee + module
- later qualifying sessions refresh issued_at and expires_at
- certification points back to the last completed training session

## Future Expansion
- skills matrix
- training records page
- expiration reporting
- manual certification adjustments
- supervisor override / recert exceptions
