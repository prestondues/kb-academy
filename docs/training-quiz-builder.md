# KB Academy Training Quiz Builder

## Purpose
Provide a quiz builder tied to training modules.

## Structure
- one quiz per module
- quiz contains many questions
- each question contains many answer options
- one or more options can be marked correct

## Current Phase
This phase builds:
- quiz metadata
- question builder
- answer option builder

## Future Phase
Later phases will add:
- quiz attempts
- trainee scoring
- pass/fail history
- certification linkage
