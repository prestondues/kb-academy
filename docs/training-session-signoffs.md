# KB Academy Training Session Sign-Offs Foundation

## Purpose
Sign-offs represent formal completion acknowledgement for a training session.

## Core Fields
- session_id
- signer_id
- signer_role
- signed_at

## Signer Roles
- trainer
- trainee

## Notes
- a session may require both trainer and trainee sign-off
- sign-off happens after required sections are complete
- later phases will add PIN verification against the employee profile
- later phases may add supervisor override or secondary review logic
