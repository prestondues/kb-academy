# KB Academy Training Time Entries and Progress

## Purpose
Track accumulated training time by user and module for time-based training paths.

## Core Model
A training time entry records:
- trainee
- module
- trainer
- optional linked session
- date
- minutes logged
- notes

## Why This Exists
Session duration alone is not enough for long-term training progress because time may be:
- logged across multiple days
- adjusted manually
- entered outside a standard runner flow
- reviewed later for audit purposes

## What This Enables
- total hours by user + module
- progress toward required hours
- remaining hours
- user training profile progress
- future certification thresholds
- future quiz + time completion logic
