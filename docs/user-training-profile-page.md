# KB Academy User Training Profile Page

## Purpose
Provide a detailed training-focused profile for a user.

## Current Capabilities
- shows user name, username, role, department, and shift
- shows current certifications
- shows completed training sessions
- shows in-progress sessions
- supports navigation from the skills matrix

## Future Expansion
- quiz history
- score history
- expiring soon certifications
- training hours rollups
- requirement gaps by department/shift/module
- supervisor notes and overrides
